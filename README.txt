Aqui les presento la Base de Datos Final v.5

----------------------------------------------------------------
FUNCIONALIDAD
-----------------------------------------------------------------
La funcionalidad va acorde a los usuarios quienes quieran usar la presente base
de datos para registros de x situacion en la que se requiera.
Esta base de datos fue hecha con el fin de ser usada en una empresa orientada 
a la agricultura (la idea de crear una base de datos surge de alli).

En el menu principal, se les presenta lo siguiente:
->Registrar
->Eliminar
->Leer
->Buscar
->Modificar
->Salir

Unicamente para moverte entre las opciones necesitas usar las letras 'i' y 'k' 
de tu teclado, con eso puedes despalzarte, y para ingresar solamente presionas
ENTER.

Dentro de Registrar tendras la opcion de registrar mediante una clave cada 
individuo a registrar, ademas de eso su nombre, apellido paterno,apellido materno,
edad, dni, telefono/celular.

Como se menciono antes, el resto de funcionalidades depende de la clave o codigo
con el que registres a cada individuo. No lo olvides. 
Por lo tanto, en eliminar tu solo tendras que colocar la clave y reafirmar la eliminacion,
en busqueda solo colocas la clave y te apareceran sus datos,
en modificar solo seras incapaza de modificar dni, el resto esta totalmente modificable,
en leer tendra cuatro opciones, una para leer todos los registro, otro para leer
solo nombre y clave de manera de bloque y otra de forma lineal, 
para asi ayudarte a recordar a que individuo estas tratando dentro del registro.
Salir es solo para salir del programa, aunque tambien puedes hacerlo presionando
la X de la ventana.

El presente proyecto aun sigue en progreso, ya que como observaron aun esta 
en la v.5, se vienen proximos arreglos.
